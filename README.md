# 🍽️ Lunch Roulette (오늘 뭐 먹지? 점심 메뉴 추천기)

대학교 1학년 Java 프로그래밍 자유 주제 과제로 제작한 **상황 맞춤형 점심 메뉴 랜덤 추천 콘솔 프로그램**입니다.  
매일 점심시간마다 반복되는 "오늘 뭐 먹지?"라는 대학생들의 현실적인 고민을 재미있게 해결하고자 기획했습니다.

<br>

## 📌 주요 기능

*   **상황별 카테고리 선택:** 사용자의 현재 기분이나 상황(스트레스, 가성비, 귀찮음 등)을 콘솔을 통해 입력받습니다.
*   **랜덤 메뉴 추첨:** 선택된 카테고리 내에서 `Math.random()`을 활용해 10가지 메뉴 중 하나를 무작위로 추천해 줍니다.
*   **입력 예외 처리:** 지정된 번호(1~3) 외의 값을 입력할 경우, 에러 메시지를 출력하고 프로그램을 안전하게 종료합니다.

<br>

## 🛠 사용 기술

*   **Language:** Java
*   **Key API:** `java.util.Scanner` (사용자 입력 제어)
*   **Core Logic:** 
    *   중첩 `switch-case`문 (조건 분기)
    *   `Math.random()` 및 타입 캐스팅 (난수 생성)

<br>
